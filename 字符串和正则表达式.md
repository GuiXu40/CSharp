## :file_folder:1.System.String类 ##
在介绍其他字符串之前，我们对String类型简单介绍，与C++一样可以使用运算符重载连接字符串：

```C#
    string s1 = "Hello";
    s1+="World";
```
使用加号就可以直接拼接字符串。还可以使用索引器找到单个字符：

```C#
    string s ="abcdefg";
    var c = s[0];//取第一个为a
```
